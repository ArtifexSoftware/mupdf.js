Working with Pages
=========================

Core API
----------------------------------

Please see the Core API methods for the `Page Class`_ for full details on the available **JavaScript** methods.


Loading a Page
----------------------------------

To load a page use the `loadPage` method as follows:

.. code-block:: javascript

    let page = document.loadPage(0)

.. note::

    Pages are zero-indexed, thus "Page 1" = index `0`.
